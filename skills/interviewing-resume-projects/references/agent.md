# Agent 项目探查参考

仅从简历陈述或候选人回答中已出现的信号选择维度。下列词语与场景仅用于定位证据缺口，不得当作候选人的项目事实。

### Agent 与确定性工作流边界
- 简历信号：Agent、自主决策、工作流、编排、多步任务
- 核验贡献：要求指出哪些步骤由模型决定、哪些由代码固定，以及本人划定边界的证据
- 深化：不确定性收益、可控性、可复现性与维护成本取舍
- 变式：任务规则稳定或合规要求提高时，哪些能力应退化为确定性流程

### 模型决策
- 简历信号：路由、意图识别、模型选择、决策节点、反思
- 核验贡献：要求还原一次决策的输入、候选动作、输出和本人用于验证正确性的记录
- 深化：概率输出、阈值、非确定性、校准与决策失败边界
- 变式：模型版本更换或输入含糊时如何保持决策质量

### 工具 Schema
- 简历信号：function calling、tool use、插件、MCP、工具注册
- 核验贡献：要求展示本人设计的参数/返回/错误结构及模型正确调用的验证证据
- 深化：约束表达、枚举、必填字段、版本兼容与最小暴露面
- 变式：工具参数新增、返回过大或模型持续生成非法参数时如何演进

### 工具错误与幂等
- 简历信号：工具重试、调用失败、重复执行、超时、外部副作用
- 核验贡献：要求说明错误分类、幂等键或去重机制，以及重复/部分成功测试
- 深化：可重试与不可重试错误、至少一次语义、副作用补偿边界
- 变式：超时后实际已成功或同一工具被并发调用时如何判定

### 规划与停止
- 简历信号：Planner、任务分解、ReAct、循环、最大步数、终止条件
- 核验贡献：要求还原计划生成、修订和停止条件，并提供一次避免空转或误停的证据
- 深化：局部/全局规划、预算、完成判据与循环检测
- 变式：目标不可达、工具持续失败或模型声称完成但产物缺失时如何停止

### 上下文
- 简历信号：上下文窗口、历史消息、摘要、上下文工程、长对话
- 核验贡献：要求说明选择、压缩和注入哪些上下文，以及效果与信息丢失证据
- 深化：相关性、时效性、位置偏差、窗口预算与污染边界
- 变式：上下文超长或相互矛盾时如何裁剪和定优先级

### 记忆
- 简历信号：长期记忆、用户画像、向量记忆、会话记忆、遗忘
- 核验贡献：要求说明记忆写入/读取/更新规则、本人实现部分及命中或误用样例
- 深化：事实与偏好、过期、冲突、删除和检索粒度取舍
- 变式：错误记忆被后续任务反复引用或用户要求删除时如何处理

### 状态持久化
- 简历信号：checkpoint、会话恢复、任务状态、持久化、断点续跑
- 核验贡献：要求给出状态模型、保存时机、恢复路径及中断恢复测试
- 深化：事件/快照、状态机、不变量、版本迁移与一致性边界
- 变式：进程在工具调用前后崩溃时如何避免丢步或重复副作用

### 重试
- 简历信号：retry、退避、容错、自愈、超时重跑
- 核验贡献：要求说明重试对象、次数/退避依据、预算和失败终态证据
- 深化：抖动、重试风暴、端到端超时及与幂等性的耦合
- 变式：依赖长期不可用或不同层同时重试时如何限流止损

### 人工审批
- 简历信号：human-in-the-loop、审批、确认、高风险操作、人工接管
- 核验贡献：要求说明审批触发规则、展示给人的证据、恢复执行方式及本人设计的控制点
- 深化：风险分级、可理解性、审批超时、拒绝和责任边界
- 变式：无人响应或审批后执行上下文已变化时如何重新校验

### 评估
- 简历信号：Agent 评测、成功率、基准、回归测试、LLM-as-judge
- 核验贡献：要求说明任务集、判定标准、基线、本人改动与误差分析
- 深化：轨迹与终态评估、确定性检查、评审偏差和统计置信度
- 变式：离线分数提升但线上完成率下降时如何定位差异

### 可观测性
- 简历信号：轨迹追踪、token 监控、工具调用日志、Agent 调试、告警
- 核验贡献：要求展示一次可串联模型、工具、状态和耗时的追踪，以及其定位价值
- 深化：trace/span 结构、敏感信息脱敏、采样、成本与质量信号
- 变式：跨轮次偶发失败或无法复现时需要保留哪些最小证据

### Prompt Injection
- 简历信号：提示词注入、越狱、不可信内容、网页/文档工具、安全防护
- 核验贡献：要求说明威胁入口、防护层、攻击样本和本人验证过的残余风险
- 深化：指令与数据隔离、间接注入、输出编码及模型防护局限
- 变式：检索文档或工具返回值包含恶意指令时如何阻断传播

### 权限
- 简历信号：沙箱、RBAC、最小权限、凭证、工具白名单、授权
- 核验贡献：要求说明主体、资源、动作、授权时点及一次拒绝或审计证据
- 深化：能力令牌、凭证隔离、权限升级、租户边界与撤销
- 变式：模型尝试调用未授权工具或用户权限在执行中变化时如何处理
